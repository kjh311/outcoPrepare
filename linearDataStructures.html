Linear Data Structure

Linked List

When an array has no more memory next to it and can't grow, another space of memory is linked to the first part and allows the array to grow.

Linked List is a data structure consisting of a group of vertices (nodes) which together represent a sequence. Under the simplest form, each vertex is composed of a data and a reference (link) to the next vertex in the sequence.

Stack

Last In, First out. Whatever you add last gets removed first.

Queue

First In, First Out. Whatever you add first gets removed first. Like a line at a supermarket.
